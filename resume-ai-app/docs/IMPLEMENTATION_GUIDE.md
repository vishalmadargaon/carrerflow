Gemini API Prompts in Detail\n\n### Prompt 1: Skill Extraction\n\n**System Context:**\n> \"You are an expert ATS (Applicant Tracking System) specialist. Analyze the provided resume and job description to extract relevant skills...\"\n\n**User Message:**\n```\nRESUME:\n[Full resume text]\n\nJOB DESCRIPTION:\n[Full job description]\n\nTASK: Return a JSON array of 20-25 key skills/tools...\nFormat: [\"Skill 1\", \"Skill 2\", ...]\nOnly return the JSON array, no explanation.\n```\n\n**Response (Example):**\n```json\n{\n  \"allSkills\": [\n    \"Python\",\n    \"AWS\",\n    \"Docker\",\n    \"Kubernetes\",\n    \"Microservices Architecture\",\n    \"RESTful APIs\",\n    ...\n  ],\n  \"matchedSkills\": [\"Python\", \"RESTful APIs\"],\n  \"missingSkills\": [\"AWS\", \"Docker\", \"Kubernetes\"],\n  \"improvementPotential\": 42\n}\n```\n\n### Prompt 2: Bullet Rewriting\n\n**System Context:**\n> \"You are a professional resume writer specializing in ATS optimization.
